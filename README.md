# Sistema de Gestión de Expedientes
📌 Descripción

Este proyecto fue desarrollado como parte de mi pasantía laboral en el Ministerio de Educación de la Provincia de Misiones (Argentina).
Desarrollado en conjunto con Facundo Pedrozo Backend Programmer.

El sistema tiene como finalidad gestionar expedientes administrativos de manera eficiente, facilitando la trazabilidad y organización de la documentación en la subsecretaría.
La aplicación fue instalada en una PC institucional dentro del Ministerio, por lo que no está disponible para uso público.

# 🛠️ Tecnologías utilizadas

Java – Lógica de negocio y backend.

TypeScript, HTML, CSS, JavaScript – Interfaz gráfica de usuario.

Arquitectura MVC – Separación clara entre modelo, vista y controlador.

# 🧩 Arquitectura y buenas prácticas

Diseño basado en el patrón MVC para garantizar orden, escalabilidad y mantenibilidad del código.

Uso de buenas prácticas de desarrollo, incluyendo separación de capas, modularización y código limpio.

Proyecto realizado en equipo, fomentando la colaboración, revisión de código y control de versiones.

# 🚀 Alcance del sistema

Registro y gestión de expedientes.

Seguimiento del estado de los documentos.

Administración segura de datos en un entorno institucional.

# ⚠️ Nota: Por motivos de seguridad institucional, no es posible mostrar imágenes adicionales del sistema ni exponer datos reales.

# 📍 Contexto profesional

Este proyecto fue desarrollado en conjunto con un compañero de pasantía como parte de la experiencia laboral en el Ministerio de Educación de Misiones.
Representa un caso real de aplicación de metodologías de desarrollo profesional, con impacto directo en la gestión administrativa del organismo.
